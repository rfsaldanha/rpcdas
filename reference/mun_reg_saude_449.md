# Municipality and health region (449) code table

A dataset containing municipality and health regions codes, following
the 449 units specification. \#'

## Usage

``` r
mun_reg_saude_449
```

## Format

A tibble with 5,597 rows and 2 variables:

- cod_mun:

  municipality 6 digits code

- cod_reg_saude:

  health region 5 digits code
